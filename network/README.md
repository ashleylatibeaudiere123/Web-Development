# Network
Network is a text-based social media application, where users can create and like posts as well as follow other users.

Video demo: To be recorded

![Network screenshot](Network.png)

## Specification
Tech used: HTML, CSS, JavaScript, Django

The homepage displays all posts created by users on the platform. After signing in, users can create their own posts, like others’ posts, and edit their own content. Users can view both their own and others’ profile pages and choose to follow other users. Posts from followed users can be accessed via the Following section.

## How to run application

Use: python manage.py runserver
