Commerce

Commerce is an online platform where users can post objects for auction and receive bids on those items.

Specification

Users can create their own listings, view active listings, and post bids. In addition, they can add the item to their watchlist and post comments on the listing. Once the bidding process closes, the winner of the auction is notified after viewing the listing.

Household Treasures

Household Treasures is an online marketplace for buying and selling items.

Specification

After logging in, users are redirected to the homepage, which shows all the items posted by users on the platform. Users can search for items through scrolling or using the search bar. After clicking on a listing’s price, users are redirected to a specific item page, where they can view all the item’s details and message the seller via email if desired. The web application further allows users to post items and track all of their active listings.

Mail

Mail is a single page application where users can send and receive email.

Specification

After signing in, users are redirected to their inbox, where ‘read’ emails appear white and unopened emails appear gray. Clicking on a specific email opens its contents while also allowing users to reply and/or archive an email. Users can also send their own emails to another registered user on the platform.

Role

Developed the front-end aspect of the website and used Javascript to send API calls to web server.

Network

Network is a text-based, social networking site.

Specification

The homepage displays all posts created by users on the platform. After signing in, users can create their own posts, like others’ posts, and edit their own content. Users can also view both their own and others’ profile pages and choose to follow other users.
